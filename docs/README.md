# docs/ — Project Documentation

This folder contains the intermediate planning documents and design specs used during the development of the Mondrian Generator.

## Files

| File | Description |
|------|-------------|
| `plan-01-mondrian-base.md` | Initial implementation plan for the single-style Piet Mondrian generator with interactive controls, seed navigation, and PNG export. |
| `plan-02-multi-artist.md` | Expansion plan covering 10 artist styles (Mondrian, van Doesburg, van der Leck, Malevich, Lissitzky, Albers, Kelly, LeWitt, Stella, Huszar), the style router, artist info modals, REFERENCES viewer, and the seed content fix. |

These documents were created as part of a subagent-driven-development workflow where one agent authored the plan and another implemented it.
